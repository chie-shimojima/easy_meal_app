# やさしい献立AI

AIを活用して、1週間分の献立を自動生成するアプリです。

忙しい日でも「無理しないで続けられる食事」をテーマに、栄養バランス・時短・やさしさを意識した献立を提案します。

## 主な機能

* 食事条件の選択（減塩・高たんぱく・時短など）
* 冷蔵庫の食材入力
* 1週間分の献立生成
* 栄養価の表示
* PDF保存機能
* 「今日は無理モード」による簡単献立提案


## 使用技術

* Python
* Streamlit
* OpenAI API
* ReportLab（PDF出力）

## 実行方法

```bash
pip install -r requirements.txt
streamlit run app.py
```
